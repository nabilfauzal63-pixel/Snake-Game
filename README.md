# 🐍 Snake Game - Grafika Komputer

Project Based Learning (PjBL) mata kuliah Grafika Komputer.

## 📌 Deskripsi
Snake Game berbasis web menggunakan:
- HTML5 Canvas
- JavaScript
- CSS
- Node.js + Express

Game ini dibuat untuk menerapkan konsep dasar grafika komputer seperti:
- Rendering
- Animasi
- Collision Detection
- Sistem Koordinat
- Transformasi Geometri

---

## 🎮 Fitur Game
- Gerakan snake menggunakan keyboard
- Collision detection
- Sistem score
- 5 level permainan
- Dynamic speed
- Wrap wall system
- Game over screen
- Mouse control

---

## 🕹️ Kontrol
| Tombol | Fungsi |
|---|---|
| ↑ | Gerak atas |
| ↓ | Gerak bawah |
| ← | Gerak kiri |
| → | Gerak kanan |
| Enter | Restart game |

---

## 🚀 Cara Menjalankan Project

### Install dependency
```bash
npm install
```

### Jalankan server
```bash
npm start
```

### Buka browser
```bash
http://localhost:3000
```
